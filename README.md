# 🏠 DomBot — семейный планировщик с ИИ

Telegram-бот для семейных задач и напоминаний.

## Что умеет
- Создать семью и пригласить родных по ссылке
- Добавить задачу в свободной форме — ИИ сам разберёт кому, что и когда
- Отправить напоминание каждому участнику лично в личку
- Кнопка Сделано прямо в напоминании
- Просмотр всех задач семьи со статусами

## Переменные окружения
BOT_TOKEN — токен от BotFather
DATABASE_URL — ссылка на PostgreSQL
OPENAI_API_KEY — ключ OpenAI
